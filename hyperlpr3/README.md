This project uses the `hyperlpr3` module from the HyperLPR repository:
https://github.com/szad670401/HyperLPR, located at `Prj-Python/hyperlpr3`.

To include it, you can clone the repo and add the folder to your Python path:
git clone https://github.com/szad670401/HyperLPR.git
cd HyperLPR/Prj-Python
export PYTHONPATH=$PYTHONPATH:$(pwd)/hyperlpr3
